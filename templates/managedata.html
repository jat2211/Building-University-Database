Manage the Data in the Student Database

Insert a table name in the corresponding box





The formatting for adding tables is as follows (exclude the parentheses):

Table Name: [table], Data: [dtype1,dtype2,...,dtypen]

For example, for adding some arbitrary major:

Table Name: Major

Add Data: 00012345,'New Major Name'

[Major]: [major_id,major_name] [int,'str']

[Student]: [stud_id,stud_name,tot_creds] [int,'str',int]

[Professor]: [prof_id,prof_name] [int,'str']

[Class]: [course_id,course_name,creds,prof_id,prof_name] [int,'str',int,int,'str']

[Major_Core]: [core_id,core_name,major_id] [int,'str',int]

[Major_Track]: [track_id,track_name,track_cred,major_id] [int,'str',int,int]

[Core_Req]: [core_id,course_id] [int,int]

[Track_Req: [track_id,course_id] [int,int]

[Majoring_in]: [stud_id,major_id] [int,int]

[Passed]: [stud_id,course_id] [int,int]
