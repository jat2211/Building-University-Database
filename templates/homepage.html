Student Database

Here, you can navigate to other pages to access different parts of the database.

Below, there is general information for the relevant data provided in the database.

Data is provided for the following links:

Majors: access to all major names and IDs

Students: access to all student names, IDs, as well as the amount of credits this student has completed

Professors: access to all professor names and IDs

Classes: access to all course names and IDs, as well as the amount of credits the course is worth and the professor information for the course

Major Cores: access to all relevant core names and IDs for a given major

Major Tracks: access to all relevant track names and IDs for a given major, if any

Core Requirements: access to all required courses for a given core

Track Requirements: access to all required courses for a given track

Student Majors: access to all student majors

Student Passed Classes: access to all courses passed by a given student

Student Core Progress: access to all courses passed by a student in a given core

Student Track Progress: access to all courses passed by a student in a given track

Change Student Majors: allows the user to change a student's major

Add Data: allows the user to add data in the database

Course Completion: allows the user to see students who have passed a given course by a given professor

New Students: allows the user to see new students i.e. students who have not completed any courses
